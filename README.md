# is-even-odd

> Checks if the input is a number and even or odd

## Functions

| Function Name | Parameters | Return Type |
| :---         |     :---:      |          ---: |
| isEven   | number     | Boolean    |
| isOdd    | number     | Boolean      |


### Sample Implementation

```javascript

import {isEven, isOdd} from "@pavan2108/is-even-odd";

isEven(10) 
// return true

isEven("10")
// return false 

isOdd(10)
// return false

```
